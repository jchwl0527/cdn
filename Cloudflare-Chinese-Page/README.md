# Cloudflare-Chinese-Page

自定义Cloudflare为中文页面，依据Cloudflare官方的界面源码进行翻译。

HTML语言编写，使用者可以轻松编辑源码中的文字。

如果网站是面向国内用户的，自定义页面可以很好的避免了部分访问者因不了解英文说明而直接关闭网页。 

网站 <https://DevHjz.com/> 

统一演示地址：
https://Chinese.Pages.Dev/
